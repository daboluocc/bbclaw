# 设计决策记录 (ADR)

记录重要的架构和设计决策。

## 记录格式

每个决策应包含：
- **编号**: ADR-XXX
- **标题**: 简洁描述
- **日期**: YYYY-MM-DD
- **状态**: 已接受 / 已废弃 / 已替代
- **背景**: 决策前的状况
- **决策**: 具体的决策内容
- **后果**: 采用后的影响
