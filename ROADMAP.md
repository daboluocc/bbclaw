# BBClaw Roadmap

## 硬件形态

**叠加态设计** - 同一设备支持 PTT / Pager 模式软件切换：

| 形态 | 描述 | 状态 |
|------|------|------|
| **PTT (Talkie-Talkie)** | 按下按键对讲，实时语音交互 | ✅ 已实现 |
| **Pager (BB机)** | 接收通知推送，轻量摘要展示 | ✅ 已实现 |
| **叠加态** | 设备同时支持两种模式，通过App切换 | 🔄 优化中 |

## Adapter 连接方案

### 当前方案：WiFi 内网模式 (已验证 ✅)

```
BBClaw 设备 ──WiFi──► 同一内网 ◄──WiFi── Adapter
     │                                    │
     │  烧录配置:                          │
     │  - WiFi SSID/Password              │
     │  - Adapter IP 地址                  │
     ▼                                    ▼
   音频/控制  ←─────── LAN ────────►  转发 → Gateway
```

**优点**: 稳定，带宽足
**缺点**: 需要烧录 WiFi 和 Adapter 地址
