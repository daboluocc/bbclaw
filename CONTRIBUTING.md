# 参与共建

感谢你对 BBClaw 的兴趣！

## 开源策略

本公开仓聚焦以下内容：

| 组件 | 状态 | 说明 |
|------|------|------|
| `firmware/` | ✅ 开源 | ESP32-S3 固件，完全开源 |
| `docs/` | ✅ 开源 | 文档完全开源 |
| `tools/` | ✅ 开源 | 本地工具、脚本与辅助资源 |

## 欢迎贡献

### ✅ 接受贡献的领域

- **固件开发** — ESP32-S3 功能的改进、Bug 修复、新功能
- **文档** — 改进文档、翻译、补充示例
- **工具脚本** — 烧录、调试脚本的改进

### ❌ 不接受 PR 的领域

- 不在本公开仓中的运行面或运维材料

如果有任何关于这些领域的建议或问题，请通过 Issue 讨论。

## 贡献流程

1. Fork 本仓库
2. 创建功能分支：`git checkout -b feature/your-feature`
3. 进行开发并确保代码风格一致
4. 提交 Pull Request
5. 我们会尽快 review 并反馈

## 行为准则

请阅读 [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) 了解社区行为规范。
