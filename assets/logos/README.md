# BBClaw Brand Assets / 品牌素材

> BBClaw is part of the [OpenClaw](https://github.com/openclaw/openclaw) ecosystem.

## 核心理念

**龙虾手拿 BB 机** — 红色像素龙虾 + 黑白 BB 寻呼机，代表 BBClaw 的产品形态（对讲机/寻呼机）和生态归属（OpenClaw 龙虾吉祥物）。

## Logo 文件

### 核心吉祥物
| File | Description | Use Case |
|------|-------------|----------|
| `bbclaw-mascot.svg` | 龙虾手拿 BB 机（像素风）64×72 | 品牌主标识 |
| `bbclaw-favicon.svg` | 龙虾 BB 机 32×32 | Favicon / tab 图标 |
| `bbclaw-icon-mini.svg` | 龙虾 BB 机 32×32 像素版 | 微型图标 |

### 完整 Logo
| File | Description | Use Case |
|------|-------------|----------|
| `bbclaw-logo-h.svg` | 横版（白色背景） | 网站顶栏、文档 |
| `bbclaw-logo-dark.svg` | 横版（深色背景） | 暗色主题 |

## 设计规范

- **吉祥物**: 红色像素龙虾 `#ff4f40`，来自 OpenClaw 官方开源素材
- **设备**: 黑白 BB 寻呼机，体现硬件产品形态
- **灵魂元素**: 龙虾的钳子夹住 BB 机，融合生态与产品
- **配色**: 红 `#ff4f40` / 黑 `#1a1a1a` / 白 `#e8e8d0` / 绿 `#88aa88`（LCD 屏）
