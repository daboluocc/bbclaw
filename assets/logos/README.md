## Design Language / 设计语言

- **产品形态**: BBClaw 是对讲机/寻呼机形态的语音交互设备，设备图标是核心标识
- **吉祥物**: 像素龙虾来自 OpenClaw 官方开源素材，体现生态归属
- **主色调**: 红色 `#ff4f40`（LED、按钮、高亮），深灰 `#1a1a1a`（机身）
- **图标风格**: 扁平化深色对讲机外形 + 红色 PTT 按键 + 像素龙虾显示屏元素
