# BBClaw Brand Assets / 品牌素材

> BBClaw is part of the [OpenClaw](https://github.com/openclaw/openclaw) ecosystem.
> The pixel lobster mascot is derived from the OpenClaw brand assets.

## Logo Versions / 标志版本

| File | Description | Use Case |
|------|-------------|----------|
| `bbclaw-logo-horizontal.svg` | Horizontal logo (mascot + text) | Website header, docs |
| `bbclaw-logo-stacked.svg` | Stacked logo (mascot above text) | App icon fallback, small UI |
| `bbclaw-logo-dark.svg` | Dark background version | Dark mode, dark themed sites |
| `pixel-lobster-16.svg` | 16×16 pixel lobster | Favicon, tiny icons |
| `pixel-lobster-32.svg` | 32×32 pixel lobster | Hi-DPI favicon, small graphics |

## Guidelines / 使用规范

- **Mascot**: The pixel lobster is shared with OpenClaw — use it to show ecosystem affiliation
- **Colors**: Primary red `#ff4f40`, outline `#3a0a0d`, claw highlight `#ff775f`
- **Text**: Use system sans-serif fonts (Helvetica/Arial) — no external font dependencies
- **Scaling**: SVG is vector — scale freely; pixel art versions (16/32) are fixed size

## Source / 来源

Pixel lobster SVG sourced from [openclaw/docs/assets/](https://github.com/openclaw/openclaw/tree/main/docs/assets).
BBClaw logos created for use across firmware, hardware docs, and promotional materials.
