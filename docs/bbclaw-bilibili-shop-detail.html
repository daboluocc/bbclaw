品牌介绍

BBClaw 是 OpenClaw 生态中的第三方硬件节点：ESP32-S3 驱动，带麦克风与扬声器、约 1.47 英寸彩屏与按键，通过 WiFi 与家中 Adapter 配对，实现本地优先的低延迟语音与状态展示。体验灵感来自对讲机与「消息感」，由同一套固件统一承载。

这是什么 BBClaw 面向 OpenClaw 链路：ASR / VAD、横屏 LVGL、PTT 与振动反馈，WiFi 连局域网 Adapter（或按文档使用公网）。本规格为 配件包，含完整外设与 3D 打印外壳，由你组装；固件发货前已烧录，无需自行编译即可配网联调。

本包包含（以实际发货清单为准）

  • 主板 / 核心板及配套连接件
  • 显示屏、音频链路器件（如 ES8311 方案相关）
  • PTT、马达等交互件
  • 3D 打印外壳（颜色 / 批次以实物为准，轻微层纹或色差属工艺范围）
  • 螺丝、排线等附件
  • 已烧录当前稳定版官方固件

另需自备

  • 可运行 Adapter 的同网段设备，配置 adapter.yaml（局域网为典型用法）
  • 公网模式：按官方文档完成账号、设备绑定与 Home Adapter 等
  • USB 线（若未随包）、螺丝刀等工具

特别声明

  • 需 用户自行组装；组装不当导致损坏一般不在质保内，以店铺规则为准。
  • 3D 件为增材制造，公差与表面纹理属正常。
  • 自行改 menuconfig、换分区或刷机，风险自担。

购买须知

  1. 非插电即聊的消费级音箱：需在同网设备运行 Adapter，并完成 WiFi、auth_token 等配置，请接受一定动手与阅读文档成本。
  2. 局域网与公网：同 WiFi IP 直连延迟更低；公网需账号、绑定与 Home Adapter 等，见官方公网说明。
  3. 固件：两版本均烧录官方固件；自行刷第三方固件导致故障，一般不保。
  4. 3D 外壳：可能有层纹、色差与微小公差；能正常装配即视为合格。
  5. 售后：请与 B 站工坊规则对齐，写明签收、质保与退换条件。
  6. 素材与品牌：第三方美术素材勿单独转售；OpenClaw / BBClaw 等使用须符合约定。
