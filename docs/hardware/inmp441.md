# INMP441 数字 MEMS 麦克风完整数据手册

## 概述

INMP441 是一款高性能、低功耗的数字输出 MEMS 麦克风，采用底部端口设计。全套方案包含 MEMS 传感器、信号调理、模数转换器、抗混叠滤波器、电源管理和标准 24 位 I²S 接口。

## 主要特性

| 特性 | 参数 |
|------|------|
| 接口 | I²S 数字接口 (24-bit) |
| SNR | 61 dBA |
| 灵敏度 | -26 dBFS (1kHz, 94dB SPL) |
| 频率响应 | 60 Hz - 15 kHz |
| 功耗 | 1.4 mA (正常) / 0.8 mA (待机) |
| PSR | -75 dBFS |
| 动态范围 | 87 dB |
| 封装 | 4.72 × 3.76 × 1 mm (LGA) |
| 工作温度 | -40°C to +85°C |
| 电源 | 1.8V - 3.3V |

## 接口定义

| 引脚 | 功能 | 描述 |
|------|------|------|
| 1 | SCK | I²S 串行时钟 |
| 2 | SD | I²S 串行数据输出 |
| 3 | WS | I²S 字选择 |
| 4 | L/R | 左/右声道选择 (低=左, 高=右) |
| 5-6, 9 | GND | 地 |
| 7 | VDD | 电源 (1.8V-3.3V) |
| 8 | CHIPEN | 芯片使能 (低=掉电, 高=使能) |

## I²S 数据格式

- 数据格式: I²S, 24-bit, 二进制补码, MSB 先出
- 时钟频率: 0.5 - 3.2 MHz
- 每个 WS 帧需要 64 个 SCK 周期
- 数据输出模式: 三态 (非输出时高阻)

## 电源管理

| 模式 | 条件 | 唤醒时间 |
|------|------|----------|
| 正常 | SCK 活动 + CHIPEN 高 | - |
| 待机 | SCK 停止 + CHIPEN 高 | 5 ms (214 时钟周期) |
| 掉电 | CHIPEN 低 | 43 ms (217 时钟周期) |

启动时间: 85 ms (218 时钟周期 @ 3.072 MHz)

## 数字滤波器特性

### 高通滤波器
- -3dB 截止: 3.7 Hz (@48kHz)
- -0.5dB 截止: 10.4 Hz
- -0.1dB 截止: 21.6 Hz

### 低通滤波器
- 通带: 0.423 × fS (48kHz = 20.3kHz)
- 通带波纹: ±0.04 dB
- 阻带衰减: >60 dB
- 群延迟: 359 µs (@48kHz)

## 频率响应

- 低频 -3dB 点: 60 Hz
- 高频 -3dB 点: 15 kHz
- 全向指向性

## PCB 设计建议

1. **电源去耦**: VDD 和地之间放置 0.1µF 电容, 尽量靠近 VDD 引脚
2. **声孔**: 建议 0.5-1.0 mm 直径
3. **SD 线**: 需要 100kΩ 下拉电阻
4. **焊接温度**: 建议使用无铅工艺, 峰值温度 260°C
