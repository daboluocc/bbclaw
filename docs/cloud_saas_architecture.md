# BBClaw 公网模式

> 随时随地连接你的"龙虾"助手

![BBClaw 公网架构](../internetSaasArch.jpg)

## 概述

公网模式让 BBClaw 设备在任意网络环境下都能使用，不再局限于家中 WiFi。

## 如何开通

### 1. 注册账户

访问 **https://bbclaw.daboluo.cc** 注册个人账户。

### 2. 绑定设备

在网页中输入设备上显示的验证码，完成硬件绑定。

### 3. 配置 HomeAdapter

在家中的 Mac mini / PC 上运行 HomeAdapter，登录同一账户即可自动关联。

### 4. 开始使用

带上 BBClaw 出门，连接任意 WiFi 或热点，即可像在家一样使用语音助手。

## 局域网 vs 公网

| | 局域网模式 | 公网模式 |
|---|---|---|
| **网络** | 同一 WiFi | 任意网络 |
| **延迟** | 极低 | 略高 |
| **距离** | 局域网内 | 无限制 |
| **适用场景** | 家中 / 办公室 | 户外、异地、跨网络 |

两种模式共享同一套固件，设备无需切换。局域网模式详见 [architecture.md](./architecture.md)。
