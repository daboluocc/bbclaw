# 电池与功耗管理基础层（P0）

## 现状

- 当前固件没有独立的电压/电量采样模块。
- 显示层没有电量状态入口。
- 下一版将加入电池、电量显示、低电量提示与后续功耗策略，因此需要先建立基础层。

## 本阶段目标

- 固定面包板默认电量采样脚为 `GPIO3`（`ADC1_CH2`）。
- 新增 `bb_power` 模块，负责：
  - ADC 初始化
  - 电池电压采样
  - 电量百分比换算
  - 向 UI 暴露统一状态
- 在 LVGL 状态栏增加电量显示入口。
- 更新 `pin_map.md`，明确 `GPIO3` 接线与分压网络。

## 当前电路基线

- 电池采样默认采用分压到 ADC 的方式。
- 默认阻值：
  - 上拉电阻 `100k`
  - 下拉电阻 `100k`
  - ADC 节点并 `100nF` 到地
- 该配置用于单节锂电池：
  - `4.2V -> 2.1V`
  - `3.3V -> 1.65V`

## 软件基线

- 空电默认 `3300mV`
- 满电默认 `4200mV`
- 第一版百分比采用线性映射，后续可替换为电池曲线表。
- 第一版仅做显示与低电量判定，不在本阶段引入：
  - 充电状态检测
  - 自动关机
  - 动态降频/休眠
  - 电源路径切换

## 后续扩展

- 增加充电检测脚 / PMIC 状态输入
- 增加 USB 供电与电池供电区分
- 增加低电量告警、自动省电策略与休眠唤醒策略
